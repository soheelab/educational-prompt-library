## 🔧 How To Use These Prompts

1. Click the prompt folder that fits your need
2. Open `prompt.xml` and copy the full text
3. Paste into Claude, ChatGPT, or Gemini
4. Follow the instructions — most prompts will ask you questions 
   before generating anything
5. Read the README for annotation and context
